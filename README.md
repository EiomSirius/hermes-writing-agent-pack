# Hermes Writing Agent Pack

5 skills instalables para Hermes Agent que automatizan tu pipeline completo de escritura.

## Contenido
1. `research_agent.skill.md` — Investiga y estructura fuentes
2. `draft_agent.skill.md` — Genera borradores profesionales
3. `editor_agent.skill.md` — Revisa gramática y estilo
4. `publish_agent.skill.md` — Formatea y publica
5. `promote_agent.skill.md` — Genera contenido promocional
