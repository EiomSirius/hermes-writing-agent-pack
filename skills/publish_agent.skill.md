# Agentic Publish Agent

Takes polished content and distributes it to the right platforms in the right format.

## When to Use
- After Editor Agent completes
- When content is ready for distribution
- For multi-platform publishing

## Procedure
1. Load final content from Editor Agent
2. Format for target platform (Medium/Gumroad/GitHub)
3. Add metadata (tags, description, cover image)
4. Generate platform-specific previews
5. Publish via API or write_file for manual posting
6. Log publication in BITACORA.md

## Verification
- [ ] Formatted correctly for target platform
- [ ] Metadata complete (title, tags, description)
- [ ] Cover image generated
- [ ] UTM tracking added
- [ ] BITACORA.md updated
