# Agentic Editor Agent

Polishes drafts to professional quality through multi-pass review.

## When to Use
- After Draft Agent completes
- Before publishing any content
- When you need a second pair of eyes

## Procedure
1. Load draft from Draft Agent
2. Pass 1: Grammar and spelling check
3. Pass 2: Style and tone consistency
4. Pass 3: Coherence and flow
5. Pass 4: Fact-check claims (web_search)
6. Generate revision log with before/after
7. Return final polished version

## Verification
- [ ] No spelling or grammar errors
- [ ] Style consistent with target audience
- [ ] All claims fact-checked
- [ ] Readability score >60
- [ ] Ready for Publish Agent
