# Agentic Draft Agent

Transforms research outlines into polished first drafts ready for editing.

## When to Use
- After Research Agent completes
- When you need a structured first draft
- For articles, guides, tutorials, or documentation

## Procedure
1. Load research data from Research Agent output
2. Define target format (article/guide/tutorial)
3. Define tone (professional/casual/technical)
4. Generate draft with: intro -> body sections -> conclusion
5. Save as markdown file
6. Flag sections needing human review

## Verification
- [ ] Draft has clear introduction
- [ ] Body sections follow logical flow
- [ ] Conclusion summarizes key points
- [ ] Tone consistent throughout
- [ ] Ready for Editor Agent
