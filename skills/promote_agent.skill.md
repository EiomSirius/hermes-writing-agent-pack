# Agentic Promote Agent

Creates and schedules promotional content for maximum reach.

## When to Use
- After Publish Agent completes
- When you need social media presence
- For content promotion

## Procedure
1. Load published content URL
2. Generate X thread (5-10 tweets summarizing key points)
3. Generate Medium article from the content
4. Create Pinterest pins with image_generate
5. Schedule posts using cronjob
6. Track engagement metrics

## Verification
- [ ] X thread ready (5+ tweets)
- [ ] At least 3 Pinterest pins generated
- [ ] Post schedule configured
- [ ] UTM tracking on all links
